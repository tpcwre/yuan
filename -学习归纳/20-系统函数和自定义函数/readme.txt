
1、引用参数   &
    在函数定义时，形参前面加上了一个&符，这就是一个引用参数

    通常用在要修改参数值的本身的时候

    注意：
        1、使用引用参数时，&要放在形参上面。
        2、如果是引用参数，传参时必须使用变量，而不能使用值。因为只有变量才有地址，普通的值是没有内存地址的。

2、自定义函数中的返回值
    通常我们写函数的时候不用echo，而是将值返回。

    函数中的值返回使用return，返回函数想要返回的值。

    return 要返回的值。

    return 返回的值，函数在哪里调用，这个值就返回到哪里去。

    注意：
        1、只要函数遇到return就直接将值返回给调用处。return语句下面的代码不会再执行。


3、局部变量
    1、函数内部声明的变量我们成为局部变量。
    2、函数的形参，就是一个局部变量，调用的时候将实参传给函数内部。

4、全局变量
    在函数外部声明的变量我们叫全局变量。可以在每个函数中使用，但是你需要做一些操作。

        global关键字

        global 变量名

    注意：
        1、可以使用global 变量名1,变量名2,.......变量名n 来声明多个全局变量。
        2、因为global在函数中产生一个指向函数外部变量的引用变量。
    
5、超全局变量$GLOBALS
    $GLOBALS，在全局中可以使用全部的变量，它是PHP自动创建的，你使用$GLOBALS['变量名']，可以在函数中使用所有的函数外的变量。

    $GLOBALS调用的是外部的变量，函数内外汇保持一致，每次使用的时候只能使用$GLOBALS这种方式.

6、静态变量
    默认的时候函数中的变量在函数执行完成后会被释放。如果我不想让他释放。我需要使用static
    关键字阻止它释放。

    格式：static 变量名;

    注意：
        1、在函数中声明的静态变量，只在第一次调用的时候声明，第二以后一看是静态变量，就先从内存里面找有没有这个变量，如果有就直接使用了，而不是再次声明。
        2、静态变量，在一个函数中多次调用，相互共享。

7、变量函数
    可变变量；将变量的值作为函数的名字加上括号来运行的函数。

8、回调函数
    通常我们在写函数的时候，是传递一个参数进去，但是有时候参数不能满足。那么我们就需要传递一个过程进去，让这个过程协助改变满足我的要求让这个过程协助改变满足我的要求。

    在函数调用的时候，在参数中传递的不是一个变量或一个值，而是一个函数的名字，这就是回调函数

    注意：
        1、回调函数，实际上他是将规则和主体函数分开了。
        2、实际上回调函数是将函数的名字以字符串的形式传递进来，然后使用变量函数的方法来运行的。


9、递归函数
    递：进去
    归：出来

    假说你背着你女朋友。你女朋友问你，我重吗？ 你就说重，因为我背上背的是整个世界。
    
    注意：
        1、递归函数，你自己就是这个函数，就是你自己调用你自己。
        2、递归函数一定要有一个条件限制，都则你会陷入一个自己调用自己的漩涡中，永不翻身，这就是个死循环。

10、加载文件
    系统函数，不用加载，系统函数能够直接用。

    我们自定义的函数也有解决方案，将函数做成全局的，将函数分类成一个一个的文件

    借助于系统指令(将文件包含进来)
        include(); 包含文件

        注意：
            1、系统里面有些像函数但不是函数的东西。  exit; exit('你想说的话');
            include "./func.php";

            2、include可以使用多次以达到包含多次、多个的目的

        include_once();包含文件，并、自动判断是否已经包含。

        注意：
            1、include_once();只是比include多了个判断，如果包含了就不再包含。
            2、建议使用include 因为include的效率比较高。

        require ("./func.php");
        require_once();

        注意：include如果在载入时候文件不存在将会给出一个警告，下面的代码还是能够执行。require在文件不存在时候给出一个致命错误，下面的代码就不能再次执行了。

        警告： 马尧 右手 骨折了。
        致命错误： 脑袋没了。

        注意：
            1、什么时候用include什么时候用reuqire，include用在一些不太重要的包含。require需要包含的是一些重要的信息。

11、数组
    标量一个变量一个值。  马尧 =====》

    数组：一组数据，数组可以存储多个值。

    113期今天的作业写1000.0遍。

    数组的最大好处是可以对数组里面的数据进行批量操作。

    1、数组的分类：
        索引数组：（叫的座位号），下标是整数。
        关联数组：（叫名字的方式），下标是字符串


        键，数组中数据对应的标识。下标。
        值，键所对应的数据

        
        键和值是相互对应的，我们通过键来访问值。

        键和值的组合，我们通常称为元素。

    2、数组的声明_1
    直接声明数组

    $var = 1;

    声明索引数组
        $var[0] = 1;
        $var['name'] = '刘硕';

    3、数组的一些特性
        注意：
            1、可以使用[ ]来声明、操作数组也可以使用{} ，但是建议使用[ ]
            2、声明数组时可以不添加下标只是使用[ ] ，默认为索引数组，默认下标从0开始。
            3、单使用[]，而不指定下表时，为索引数组，索引下表的值将为出现过的最大的下标+1。
