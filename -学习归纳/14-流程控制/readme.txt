    1、多向分支结构(elseif子句)
    >=90 优秀
    >=80 良好
    >=70 一般
    >=60 及格
    < 60 不及格

        格式：
            if(条件表达式1){
                代码段1
            }else if(条件表达式2){
                代码段2
            }.....
            else{
                上面的条件都不成立的时候执行的代码段
            }

            如果条件表达式1返回true就执行代码段1，条件表达式2返回true就执行代码段2....if里面的语句都不成立的时候执行else从句里面的内容

            注意：
                1、else从句可以省略
                2、else if 也可以写成 elseif，但是建议你写成else if
                3、if条件从上往下执行，执行了一处大括号内的代码块之后其他的条件就不会再检查、执行
                4、如果就一条语句可以不加大括号，但是建议加上。
                5、通常用来做范围条件的判断

            ......
            default:
                语句n

        }

        switch()里面的变量如果和值1能够匹配的话就执行语句1，如果一直到最后都没有匹配上那么将执行default里面的东西。default也是可选的他通常用来接收条件都不相等的情况，else

        注意：
            1、switch()一般来说放的是变量（字符串或整数）或常量
            2、switch的作用通常用来判断匹配某个值。
