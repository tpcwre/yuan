今日课程：
    1、数组的特性（书接上文）
        1）、声明数组时，索引数组下标可以不连续。指定什么就是什么。
        2）、负数可以作为下表，但是没有太大意义。
        3）、声明关联数组时如果下标不加引号来包含下表，会当作常量来使用。如果连这个常量都没有才会转换为字符串
        4）、如果下标是null，那么下标将为空字符串，如果下标为小数将会舍掉小数位。
        5）、索引下标和关联下标可以放在同一个数组中，但是我们不经常用。
        6）、数组的关联下标和索引下表不会相互影响。关联下标不会影响索引下标的排列规则。
        
    2、数组的声明_2
    使用array()来声明数组
    注意：
        1、array()没有指定下标，默认是索引数组，从0开始
        2、array()可以指定下标，使用=>操作符来进行声明，索引和关联都这样做

    3、数组的声明_3
    使用[]来直接声明，[]是array()的简化方式。

    注意：
        1、这种写法得是php 5.4版本以后可以这样使用。
        2、新版本中可以使用一种方式来获取。

        <?php
            //$var = array('a'=>'1',1=>'b');
            //$var = ['a'=>1,1=>'b'];

            //var_dump($var);
            function test(){
                return array(1=>'a',2=>'b');
            }

            /*$var1 = test();
            echo $var1[1];*/

            echo test()[1];
        ?>

    4、删除数组中的元素
        unset();直接在内存中释放，下标释放。
        null值：删除指定元素下标的值，设置为null ，下标保留

        注意：
            1、单独使用[]时，不指定下标的时候，索引下标的值将为出现过得最大的下标+1。

            谈恋爱：   1任：分手了    2任：分手了  3任：分手了。 n+1任：结婚了

    5、多维数组的声明和使用
        多维：维度上面的区分，层。 就是在数组的值是数组。可以是关联数组、索引数组

        注意：
            1、多维数组尽量控制在3层以内。
            2、使用多维数组一层一层的往下找。

        每个数组的元素的值，也是数组。

    6、数组的遍历_for
        注意：如果直接输出一个数组将会将数组变为array（自动类型转换的）

        注意：
            1、count()函数用来计算数组中元素的个数
            2、for循环遍历的数组只能是索引数组。
            3、下标必须是连续的，最好从0开始。

    7、   数组的遍历_foreach
    格式：
        foreach(数组 as 自定义的变量){

        }

    注意：
        1、数组中有多少个元素foreach就循环多少次（自动的）。
        2、第一次循环是将第一个元素的值赋给了自定变量，第二次也就将第二元素的值赋给了自定义的变量，以此类推。
        3、不管是索引数组还是关联数组都可以使用

    格式：
        foreach(数组 as 自定义变量1=>自定义变量2){
        }

        注意：
            1、自定义变量1是下标，自定义变量2当前元素的下标所对应的值。
            2、foreach和for，foreach的功能强大一些，但是不如for运行的快。因为for循环取值是用的PHP语言中本身带的语法。$array[0]
            3、在用的时候如果索引数组下标为0，并且是连续的那么就使用for循环，其他的时候使用foreach
